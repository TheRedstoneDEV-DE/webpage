## Hardware

### Homelab

#### Server 1: The NAS

  • OS: Fedora Server 40
  • CPU: Intel N100
  • GPU: integrated [no dGPU]
  • Memory: 8GB [DDR5]
  • Networking: 4x 2.5GbE [Onboard]
  • Drive configuration:
    • 2x 4TB WD HDDs [RAID-1] [HOTSWAP]: used for backups
    • 2TB WD NVMe SSD: used for my Steam library
    • 256GB NVMe SSD: boot drive
    • 512GB SATA SSD [HOTSWAP]: service data
  • Services:
    • Pi Hole
    • Navidrome
    • Jellyfin
    • (ethernet switch / firewall for the other servers)
    • sshfs

  • OS: Ubuntu Server 22.04
  • CPU: AMD Ryzen 5 3600
  • GPU: NVidia RTX 3060 12GB
  • Memory: 32GB [DDR4 3200]
  • Networking: 1x 1GbE [Onboard]
  • Drive configuration:
    • 512GB WD NVMe SSD: boot drive
    • 512GB SATA SSD: VM data
  • Services:
    • Libvirtd
    • Game-servers

#### Server 3: The Remote Workstation

  • OS: Nobara 40
  • CPU: AMD Ryzen 5 2600
  • GPU: NVidia RTX 3060 12GB
  • Memory: 16GB [DDR4 2666]
  • Networking: 1x 1GbE [Onboard]
  • Drive configuration:
    • 60GB SATA SSD: boot drive
